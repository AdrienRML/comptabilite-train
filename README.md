# Comptabilite Train

## Project Overview
Comptabilite Train is a project designed to streamline the accounting processes for train schedules and related financial transactions. This application aims to provide a user-friendly interface for managing train data, tracking expenses, and generating financial reports.

## Features
- Train schedule management
- Expense tracking for train operations
- Financial reporting capabilities

## Setup Guide

### Prerequisites
Before you begin, ensure you have met the following requirements:
- You have a working installation of Node.js.
- You have npm (Node Package Manager) installed.

### Installation Steps
1. **Clone the repository**  
   Run the following command to clone the project repository:
   ```bash
   git clone https://github.com/AdrienRML/comptabilite-train.git
   ```

2. **Navigate to the project folder**  
   ```bash
   cd comptabilite-train
   ```

3. **Install dependencies**  
   Run the following command to install the project dependencies:
   ```bash
   npm install
   ```

4. **Run the application**  
   After installation, you can run the application using:
   ```bash
   npm start
   ```

5. **Access the application**  
   Open your browser and navigate to `http://localhost:3000` to use the application.
